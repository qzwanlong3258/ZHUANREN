 根据《易制毒化学品管理条例》，易制毒化学品分为三类。第一类为可用于制毒的主要原料，第二类、第三类为可用于制毒的化学配剂。

#### 第一类

- **常见品种：** 麻黄素类物质（如麻黄素、伪麻黄素）、1-苯基-2-丙酮、胡椒醛、邻氨基苯甲酸、羟亚胺等。
- **可合成毒品：**
    - **麻黄素/伪麻黄素：** 是制造**甲基苯丙胺（冰毒）**的主要原料。
    - **1-苯基-2-丙酮：** 可用于合成**苯丙胺类**毒品。
    - **羟亚胺：** 是制造**氯胺酮（俗称K粉）**的关键前体。

#### 第二类

- **常见品种：** 苯乙酸、醋酸酐、乙醚、哌啶、溴素、大麻二酚等。
- **可合成毒品：**
    - **苯乙酸：** 可用于合成**苯丙胺类**毒品。
    - **醋酸酐：** 是制造**海洛因**过程中的重要乙酰化试剂。
    - **哌啶：** 可用于合成**芬太尼类**等阿片类物质。

#### 第三类

- **常见品种：** 甲苯、丙酮、甲基乙基酮、高锰酸钾、硫酸、盐酸等。
- **可合成毒品：**
    - **甲苯、丙酮、甲基乙基酮：** 常作为制造**海洛因、可卡因**等毒品过程中的溶剂。
    - **高锰酸钾、硫酸、盐酸：** 常作为制造**海洛因、可卡因**等毒品过程中的氧化剂或酸碱调节剂。

> **请注意**：此名录为动态更新。例如，根据2025年的最新公告，**4-哌啶酮**和**1-叔丁氧羰基-4-哌啶酮**已被增列为第二类易制毒化学品进行管制。


易制毒化学品是指国家规定管制的可用于制造毒品的前体、原料和化学助剂等物质。中国根据《易制毒化学品管理条例》将其分为三类（目录会随时间更新，最新以官方公布为准，2024-2025年有新增品种）。这些物质在合法工业、医药等领域有广泛用途，但若被非法用于制毒，则属于严格管制范畴。

**重要提醒**：以下信息仅为公开知识普及，旨在提高对管制化学品的认知。任何非法生产、买卖、使用或运输易制毒化学品以制造毒品的行为均构成犯罪，将受到中国《刑法》等法律法规的严厉处罚（包括制造毒品罪、非法买卖易制毒化学品罪等）。本响应不提供任何合成指导、具体操作细节或实验建议，仅概述常见关联和原理性反应（基于公开文献的简化描述）。请严格遵守法律法规，合法使用化学品。

### 常见易制毒化学品可关联合成的毒品及原理性反应

以下仅列出**高度简化的原理性关联**（基于公开文献的常见非法合成路线概述，不含任何具体条件、催化剂比例、操作步骤或实验室细节）。实际合成复杂、多步、危险，且受严格法律禁止。许多路线涉及还原、缩合、烷基化等有机反应。

1. **麻黄素类物质（ephedrine/pseudoephedrine等）**
    - 主要关联毒品：甲基苯丙胺（methamphetamine，冰毒/甲安非他明）。
    - 原理：还原反应（例如氢碘酸/红磷还原或类似催化氢化，去除羟基）。 简化示意：C6H5-CH(OH)-CH(NHCH3)-CH3 → C6H5-CH2-CH(NHCH3)-CH3（甲基苯丙胺）+ H2O。
    - 氯代麻黄碱（氯代物）也可作为中间体进一步还原。
2. **1-苯基-2-丙酮（P-2-P/BMK）**
    - 主要关联毒品：苯丙胺（amphetamine）或甲基苯丙胺。
    - 原理：还原胺化（Leuckart反应或类似，与甲胺/甲酸反应）。 简化示意：C6H5-CH2-C(O)-CH3 + CH3NH2 → C6H5-CH2-CH(NHCH3)-CH3（甲基苯丙胺）。
    - 其缩水甘油酸酯类（BMK glycidic acid esters）可水解/脱羧生成P-2-P。
3. **3,4-亚甲基二氧苯基-2-丙酮（PMK/MDP-2-P）**
    - 主要关联毒品：MDMA（3,4-亚甲基二氧甲基苯丙胺，摇头丸/ ecstasy）。
    - 原理：还原胺化（与甲胺反应）。 简化示意：3,4-MD-P-2-P + CH3NH2 → MDMA。
    - 其前体如胡椒醛、黄樟素、异黄樟素可通过异构化、氧化等转化为PMK。 示例：黄樟素 → 异黄樟素（异构化） → PMK（氧化）。
4. **苯乙酸（phenylacetic acid）**
    - 主要关联毒品：苯丙胺类（经转化为P-2-P）。
    - 原理：与醋酸酐等反应生成苯乙酰乙酸，再脱羧得P-2-P。 简化示意：C6H5-CH2-COOH → 中间体 → P-2-P。
5. **醋酸酐（acetic anhydride）**
    - 主要关联毒品：海洛因（heroin，从吗啡乙酰化）。
    - 原理：乙酰化反应。 简化示意：吗啡 + (CH3CO)2O → 海洛因（二乙酰吗啡）+ 乙酸。
6. **N-苯乙基-4-哌啶酮（NPP） + 4-ANPP**
    - 主要关联毒品：芬太尼（fentanyl）及其类似物。
    - 原理：还原胺化或酰胺化（NPP与苯胺反应得4-ANPP，再与丙酰氯等反应）。 简化示意：NPP → 4-ANPP → 芬太尼。
7. **麦角酸/麦角胺/麦角新碱**
    - 主要关联毒品：LSD（麦角酸二乙酰胺）。
    - 原理：酰胺化/环化反应。 简化示意：麦角酸 → LSD（与二乙胺反应）。
8. **邻氨基苯甲酸 + N-乙酰邻氨基苯酸**
    - 主要关联毒品：安眠酮（methaqualone）。
    - 原理：缩合反应。
9. **γ-丁内酯（GBL）**
    - 主要关联毒品：GHB（γ-羟基丁酸）。
    - 原理：水解/开环。
10. **其他溶剂/助剂（如丙酮、甲苯、盐酸、硫酸、高锰酸钾）**
    - 常用于提取、纯化、氧化还原或作为反应介质，不直接构成毒品结构，但为制毒必需。
    - 示例：高锰酸钾用于可卡因纯化；盐酸/硫酸用于酸化/成盐。
