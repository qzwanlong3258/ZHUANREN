在化工装置停工检修条件的联合检查确认中，**施工主管部门**（通常为设备部、工程部或机动部）扮演着“总把关人”的角色。

根据行业规范和安全管理要求，施工主管部门的核心职责可以概括为：**“审资质、核方案、查设备、管准备”**。

具体来说，施工主管部门在联合检查中必须落实以下**四项关键工作**：

### 🛡️ 1. 审核承包商资质与人员资格（“审资质”）

这是准入的第一道门槛，施工主管部门必须确保“正规军”进场。

- **单位资质核查**：检查承包商是否具备相应的施工资质（如机电安装、压力容器维修等）和有效的营业执照。
- **HSE业绩审查**：查阅承包商过去的安全业绩证明，确保其安全记录良好。
- **人员资格核对**：
    - **特种作业人员**：必须逐一核对焊工、电工、起重工等特种作业人员的操作证，确保**人证合一**且在有效期内。
    - **专职安全员**：检查承包商是否配备了具备资质的专职安全管理人员，且数量满足现场监护需求。

### 📝 2. 审批检修方案与HSE方案（“核方案”）

这是技术安全的核心，施工主管部门需对施工方案的可行性负责。

- **方案审批**：组织对**检修施工方案**进行审核。重点检查施工步骤是否合理、进度安排是否科学。
- **HSE方案审核**：审核承包商提交的危害识别（JSA）和安全技术措施。确保方案中包含了针对本项目风险的具体防范措施（如防火、防坠落、防中毒措施）。
- **应急预案**：确认承包商是否制定了现场处置方案或应急预案，并已报备。

### 🏗️ 3. 检查特种设备与工器具（“查设备”）

防止“带病”设备进入装置区，从源头消除物的不安全状态。

- **特种设备安全**：检查进场的起重机械、叉车、升降车等特种设备是否具有有效的检验合格证书。
- **工器具检查**：
    - **防爆要求**：在易燃易爆区域使用的工具（如扳手、锤子）必须是防爆工具。
    - **电气安全**：检查临时用电设备（配电箱、电缆）是否符合规范，漏电保护是否灵敏。

### ✅ 4. 确认施工准备情况（“管准备”）

这是联合检查的最终确认环节，确保“万事俱备”。

- **施工准备审核**：综合检查承包商的物资准备、人员到位情况、现场临时设施搭建情况（如工棚、仓库）。
- **安全协议与保险**：确认是否已签订**施工HSE合同**（或安全协议），并检查是否已按规定购买了工伤保险和**安全生产责任险**。
- **安全抵押金**：根据企业管理制度，检查是否已缴纳安全风险抵押金。

### 📌 总结：施工主管部门的“一票否决权”

在联合检查中，如果施工主管部门发现以下任一情况，应行使**一票否决权**，拒绝签字交出：

1. **资质不符**：承包商无资质或超范围经营。
2. **方案未批**：施工方案或安全措施未经审批。
3. **人员无证**：特种作业人员无证上岗。
4. **设备带病**：进场特种设备无检验合格证。

只有施工主管部门确认上述内容全部合格，装置才能正式移交给承包商进行检修作业。
