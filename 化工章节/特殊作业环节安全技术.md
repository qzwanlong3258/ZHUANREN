1、作业前的安全防护措施

| 工艺处置及分析       | ① 与生产系统断开、隔离（不应用水封或关闭阀门代替盲板）。<br>② 吹扫、清洗、置换后，打开人孔（自上而下依次打开）。<br>③ 对动火点周围可燃物、电缆桥架、孔洞、窨井、地沟、水封设施、<br>污水井等清理或封盖。<br>④ 对于动火点周围15m 范围内有可能泄漏易燃、可燃物料的设备<br>设施，应采取隔离措施。<br>⑤ 在管道、储罐、塔器等设备外壁上动火，应在动火点10m 范围<br>内进行动火气体分析，同时还应检测设备内气体含量。<br>　(a) 气体分析取样时间与动火作业开始时间间隔不应超30min。<br>　(b) 特级、一级动火作业中断时间超过30 min，二级动火作业中<br>　　　断时间超过60 min，应重新进行气体分析。<br>　(c) 每日动火前均应进行气体分析。<br>　(d) 特级动火作业期间应连续进行监测。<br>　(e) 被测气体或蒸气的爆炸下限≥4%时，其被测浓度应≤0.5%；<br>　　　被测气体或蒸气的爆炸下限＜4%时，其被测浓度应≤0.2%。<br>   (f）在较大的设备内动火，采样点必须选择有代表性的上、中、下<br>（左、中、右）三个点进行检测；设备内的气体比空气重时，应在<br>底部采样；设备内的气体比空气轻时，应在上部采样。<br>（g）在生产、使用、储存氧气的设备上进行动火作业时，设备内氧<br>含量应≤23.5%。 |
| ------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 动火环境的检<br>查确认 | ① 作业前，对动火点周围下水系统储油进行冲洗并封堵。<br>② 下水井及地漏系统：2层的石棉布覆盖＋5cm 厚的细土封堵，或<br>水泥抹死，无存油的地沟要灌满水。<br>准备好消防器材。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| 作业票的办理        | ① 一张“动火安全 作业票”只限一处动火，实行一处（一个动火<br>地点）、一证（“动火安全作业票”）、一人（动火监护人）。<br>② 在盛装或输送可燃气体、可燃液体、有毒有害介质或其他重要的<br>运行设备、容器、管线上进行焊接作业时，设备管理部门必须对施<br>工方案进行确认，对设备、容器、管线进行测厚，并在动火安全作<br>业票上签字。<br>③ 施工动火作业涉及其他管辖区域时，由所在管辖区域单位领导审<br>查会签，并由双方单位共同落实安全措施，各派一名动火监护人。                                                                                                                                                                                                                                                                                                                                                                    |
| 安全措<br>施交底    | ① 基层单位必须向施工单位进行现场检查交底。<br>② 施工单位作业负责人应向施工作业人员进行作业程序和安全措施<br>交底，并指派作业监护人。<br>③ 交底的内容：<br>a、作业现场和作业过程中可能存在的危险、有害因素及采取的具体<br>安全措施；<br>b、会同作业单位组织作业人员到作业现场，了解和熟悉现场环境，<br>进一步核实安全措施的可靠性，熟悉应急救援器材的位置及分布；<br>c、涉及断路、动土作业时，应对作业现场的地下隐蔽工程进行交底。<br>d、所有参加交底的人员必须履行签字手续，基层单位班组、交底人、<br>作业人员、作业监护人各留一份，并记录存档。                                                                                                                                                                                                                                                                                                                  |
3、作业过程中的安全防护措施

| 三不动火         | 没有经批准动火安全作业票不动火；<br>动火监护人不在现场不动火；<br>安全管控措施不落实不动火。                                                                                                                                                                                                                                                                                                                                                 |
| ------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 对作业监护人的要求    | ① 应参加企业培训，获取动火监护人资格证书，做到持证上岗。<br>② 监护过程中应佩戴明显标志，如挂牌或者穿反光马甲等。<br>③ 动火过程中，不得离开现场。确需离开时，由监护人收回“动火作业证”，暂停动火。<br>④ 发现异常情况，立即停止动火。当作业内容发生变更时，应立即停止作业，“动火作业证”同时废止.                                                                                                                                                                                                                                        |
| 对作业人员的要求     | ① 特种作业人员应有相应的工作资格证书。<br>② 施工单位作业负责人、安全管理人员具有安全资质证书。<br>③ 认识安全警示标志，具备逃生技能。<br>④ 会使用常用消防器材。<br>⑤ 有权拒绝违章指挥的指令。<br>⑥ 穿戴好防护用品，如防护手套、防护口罩、护目镜和安全带等.<br>记忆：2 证 2 防，拒绝逃生                                                                                                                                                                                                                                   |
| 动火作业过程中的管理要求 | ① 距离<br>距动火点≤30m 严禁排放各类可燃气体；<br>距动火点≤15m 严禁排放各类可燃液体；<br>距动火点≤10m 不应进行可燃性粉尘清扫作业，动火点上方及下方还不应同时进行可燃溶剂清洗或喷漆作业；<br>在厂内铁路沿线≤25m 动火作业时，如遇装有危险化学品的火车通过或停留时，应立即停止作业。<br>② 装置停工吹扫期间，严禁一切明火作业。<br>③ 特级、一级动火安全作业票有效期不应超过8h；二级动火安全作业票有效期不应超过72h。<br>④ 遇五级风以上（含五级风）天气，禁止露天动火作业；因生产确需动火，动火作业应升级管理。<br>⑤ 特级动火作业应采集全过程作业影像，且作业现场使用的摄录设备应为防爆型。<br>⑥ 在受限空间内进行动火作业。临时用电作业时，不允许同时进行刷漆、喷漆作业或使用可燃溶剂清洗等其他可能散发易燃气体、易燃液体的作业。 |
4、作业结束后的安全防护措施
（1）工完、料净、场地清。
（2）监护人确认现场满足安全条件后，在“动火安全作业票”的“完工验收”栏中签字。
（3）如果第二天继续施工，施工余料或机具可暂放在现场，不得占用消防通道，不得放在巡检通道、过桥台阶上。

二、受限空间作业安全技术
1、作业前的安全防护措施


| 工艺处置           | ① 清洗与置换，如加热蒸煮等。<br>② 隔离：<br>　　a、与受限空间连通的可能危及安全作业的管道应采用加盲板或拆除一段管道的方式进行隔离（不应用水封或关闭阀门代替盲板）。<br>　　b、与受限空间连通的可能危及安全作业的孔、洞应进行严密封堵。<br>　　c、对作业设备上的电器电源，应采取可靠的断电措施，电源开关处应上锁并加挂警示牌。<br>③ 通风：<br>　　a、打开人孔、手孔、料孔、风门、烟门等与大气相通的设施。<br>　　b、必要时，可采用强制通风或管道送风。<br>　　c、在忌氧环境中作业，通风前应对作业环境中与氧化性物质相抵的物料采取卸放、置换或清洗合格的措施。<br>④ 气体环境要求：<br>a、作业前 30 min 内，对受限空间进行气体检测。<br>b、容积较大的受限空间，应对上、中、下（左、中、右）各部位进行<br>检测分析。<br>c、涂刷具有挥发性溶剂的涂料时，应采取强制通风措施。<br>d、作业中断时间超过 60 min 时，应重新进行气体检测分析。<br>e、不应向受限空间充纯氧气或富氧空气。<br>f、检测人员进入或深入受限空间检测时，应佩戴符合规定得个体防护<br>装备。 |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 气体检测           | ① 氧气含量为 19.5%～21%，在富氧环境下不应大于 23.5%。<br>② 首次采样分析，必须使用色谱仪进行分析。<br>③ 作业时，作业现场应配置移动式气体检测报警仪，连续检测受限空<br>间内可燃气体、有毒气体及氧气浓度，并 2h 记录 1 次。<br>④ 可燃气体、蒸气浓度要求应符合动火分析合格判定指标的规定。（大<br>45，小 42）                                                                                                                                                                                                                                                                                                                                                |
| 作业票的办理         | ① 受限空间施工作业单位作业负责人，应持有施工任务单，到设施所属单<br>位办理“受限空间安全作业票”。<br>② 设施所属单位安全负责人和领导要对作业程序和安全措施进行确认<br>后，方可签发“受限空间安全作业票”，并指派作业监护人。施工单位<br>作业负责人应向作业人员进行作业程序和安全措施的交底，并指派<br>作业监护人。<br>③ 受限空间作业时，应将相关的受限空间安全作业票、施工方案、应<br>急预案、气体连续监测记录、人员及工器具出入登记等文件粘挂在现场。                                                                                                                                                                                                                                                                                    |
| 安全<br>措施<br>变更 | ① 基层单位必须向施工单位进行现场检查交底。<br>② 施工单位作业负责人应向施工作业人员进行作业程序和安全措施交底，并指派作业监护人。<br>③ 交底的内容：<br>　a、作业现场和作业过程中可能存在的危险、有害因素及采取的具体安全措施；<br>　b、会同作业单位组织作业人员到作业现场，了解和熟悉现场环境，进一步核实安全措施的可靠性，熟悉应急救援器材的位置及分布情况；<br>　c、涉及断路、动土作业时，应对作业现场的地下隐蔽工程进行交底。<br>④ 所有参加交底的人员必须履行签字手续，基层单位班组、交底人、作业人员、作业监护人各留一份，并记录存档。                                                                                                                                                                                                                                          |
| 作业人员<br>员的要求   | ① 作业人员应充分了解作业内容，熟知作业中的危害因素和相应的安<br>全措施，持有审批同意的“受限空间安全作业票”方可施工作业。<br>② 对违反制度的强令作业、违章指挥、安全措施不落实、作业监护人<br>不在场等情况有权拒绝作业。<br>③ 个体防护：<br>a、缺氧或有毒的受限空间经清洗置换仍达不到要求的，应佩戴隔绝<br>式呼吸防护装备，并正确连接救生绳。<br>b、在受限空间内从事清污作业，应佩戴隔绝式呼吸防护装备，并正确<br>连接救生绳。<br>c、易燃易爆的受限空间经清洗或置换仍达不到要求的，应穿防静电工<br>作服及工作鞋，使用防爆工具具。<br>d、在受限空间内作业时，应配备相应的通信工具。<br>e、有酸碱（腐蚀），防酸碱（腐蚀）；电焊，绝缘鞋；有噪声，耳塞、<br>耳罩；有粉尘，防尘口罩；高温，高温防护用品；低温，低温防护用品。                                                                                                                                    |
| 监护<br>人的要求     | ① 作业期间，严禁离岗。<br>② 熟悉作业区域的环境和工艺情况，可判断和处理异常情况的能力、<br>懂急救知识。<br>③ 发现安全措施不落实或不完善时，要及时制止作业活动。<br>④ 监护人一般应有2人，携带移动式气体检测报警仪器、通信设备、<br>救援设备。<br>⑤ 对进入受限空间的人员及其携带的工器具种类、数量进行登记，作业 <br>完毕后再依次进行清点，防止遗漏在受限空间内。                                                                                                                                                                                                                                                                                                                             |
2、作业过程中的安全防护措施

| 三不进入  | 未持有经批准的受限空间作业票不进入；<br>安全措施不落实到位不进入；<br>监护人不在场不进入。                                                                                                                     |
| ----- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 安全作业票 | 当受限空间状况改变时，作业人员应立即撤出现场。处理后需重新<br>办理受限空间安全作业票方可进入。受限空间安全作业票有效期<br>不应超过24h。                                                                                             |
| 照明电压  | 进入金属容器（炉、塔、釜、罐等）和特别潮湿、工作场地狭窄的<br>金属容器内作业，照明电压≤12V。<br>受限空间作业环境原来盛装可燃性液体、气体等介质的，应使用防<br>爆电筒或电压≤12V的防爆安全灯行，灯行变压器不得放在容器内<br>或容器上；作业人员应穿戴防静电服装，使用防爆工具、严禁携带<br>手机等非防爆通信工具。 |
| 通风    | 为保证受限空间内空气流通和人员呼吸需要，可采用自然通风。必要时采取强制通风，严禁向内充氧气                                                                                                                         |
| 监护人   | 受限制空间作业监护人严禁进入受限空间内，受限空间内的作业人员发生中毒、窒息的紧急情况，监护人严禁未佩戴防护用具即进入受限空间内，应迅速与其他人员联系，抢救人员必须佩戴隔离式防护面具进入受限空间，并至少有１人在受限空间外部负责联络工作。                                                 |
| 安全    | 施工的氮气瓶、乙炔瓶严禁带入受限空间内。<br>受限空间内动火作业时，严禁同时进行刷漆、防腐作业。                                                                                                                     |
