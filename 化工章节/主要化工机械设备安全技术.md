化工机械设备安全要求：
（1）应当有足够的强度；（2）应当做到密封可靠；（3）必须具备配套的安全保护装置；（4）应当具有很强的适用性；（5）规范运行操作和检维修
管理。（记忆：宝强蜜室孕检）
一、压缩机

| ------ | --------------------------------------------------------------------- |
| 按排气压力分 | 低压（0.3 ～ 1.0 MPa）　中压（1.0 ～ 10 MPa）<br>高压（10 ～ 100 MPa）、超高压（＞ 100 MPa） |
| 按结构型式分 | 容积式    回转式（包括螺杆式、滑片式、罗茨式）                                             |
|        | 容积式    往复式（包括活塞式、隔膜式）                                                 |
|        | 速度式 离心式、轴流式、喷射式、混流式                                                   |

三、安全阀
1、安全阀的安装和使用

| 安装位置  | ① 设备或管道上的安全阀竖直（铅直）安装。<br>② 一般安装在靠近被保护设备，安装位置易于维修和检查。<br>③ 蒸汽安全阀装在锅炉的锅筒、集箱的最高位置，或装在被保护设备液面以上气相空间的最高处。<br>④ 液体安全阀装在正常液面的下面。                                                                  |
| ----- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 进出口管道 | （1）安全阀的进口管道直径≥安全阀的进口直径。<br>（2）安全阀的出口管道直径≥安全阀的出口直径。<br>（3）如果几个安全阀共用一条进口管道时，进口管道的截面积≥这些安全阀的进口截面积总和。<br>（4）安全阀出口的排放管上如果装有消声器，必须有足够的流通面积。<br>（5）安全阀的进出口管道一般不允许设置截断阀，必须设置截断阀时，需要加铅封，并且保证锁定在全开状态 |
| 选用    | ① 安全阀适用于清洁、无颗粒、低黏度的流体。<br>② 全启式安全阀适用于排放气体、蒸汽或者液体介质，微启式安全阀一般适用于排放液体介质，排放有毒或者可燃性介质时必须选用封闭式安全阀。                                                                                               |
| 运行    | 锅炉运行中，安全阀需要定期进行手动排放试验，锅炉停止使用后又重新启用时，安全阀也需要进行手动排放试验。                                                                                                                                        |
2、安全阀的定期检查

| 停止使用 | 阀瓣和阀座密封面损坏，已经无法修复           | 报废      |
|------|-----------------------------|---------|
|      | 导向零件锈蚀严重，已经无法修复             | 报废      |
|      | 调节圈锈蚀严重，已经无法进行调节            | 报废      |
|      | 弹簧腐蚀，已经无法使用                 | 报废      |
|      | 附件不全而无法配置                   | 报废      |
| 停止使用 | 历史记录丢失                      | 停止使用并更换 |
|      | 选型不当                        | 停止使用并更换 |
| 定期校验 | ① 安全阀定期校验，一般每年至少一次。         |         |
|      | ② 经解体、修理或更换部件的安全阀，应当重新进行校验。 |
四、爆破片安全装置
1、爆破片单独使用

| 适用场合  | ① 容器内压力迅速增加，安全阀来不及反应的。  <br>	② 设计上不允许容器内介质有任何微量泄漏的。  <br>	③ 容器内介质产生的沉淀物或黏着胶状物有可能导致安全阀失效的。  <br>	④ 由于低温的影响，安全阀不能正常工作的。  <br>	⑤ 由于泄压面积过大或泄放压力过高（低）等原因安全阀不适用的。 |
| ----- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 特殊    | 移动式压力容器的相关标准有特殊规定的，如长管拖车和管束式集装箱等被保护承压设备可使用爆破片安全装置作为单一超压泄放装置。                                                                                                |
| 不适用场合 | 经常超压或温度波动较大的被保护承压设备，不应单独使用爆破片安全装置作为超压泄放装置。                                                                                                                  |
2、爆破片安全装置与安全阀组合使用
（1）爆破片串联在安全阀入口侧

| 设置<br>情形 | ① 为避免因爆破片的破裂而损失大量的工艺物料或盛装介质的。<br>② 安全阀不能直接使用场合（如介质腐蚀、不允许泄漏等）的。<br>③ 移动式压力容器中装运毒性程度为极度、高度危害或强腐蚀性介质的.                                                                                                                                                           |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 设置<br>要求 | ① 爆破片与安全阀组合装置的泄放量应≥被保护承压设备的安全泄放量。（组合装置泄放量＝单个安全阀额定泄放量 × 0.9）<br>② 爆破片公称直径应≥安全阀入口侧管径，并应设置在距离安全阀入口侧5倍管径内，且安全阀入口管线压力损失（包括爆破片导致的）应不超过其设定压力的3%。<br>③ 爆破片爆破后的泄放面积应＞安全阀的进口截面积。<br>④ 爆破片在爆破时不应回产生碎片、脱落或火花，以免妨碍安全阀的正常排放功能。<br>⑤ 爆破片安全装置与安全阀之间的腔体应设置压力指示装置、排气口及合适的报警指示器。 |
(2) 爆破片安全装置串联在安全阀的出口侧

| 设置情形 | （1）若安全阀出口侧有可能被腐蚀或存在外来压力源的干扰时，应在安全阀出口侧设置爆破片安全装置，以保护安全阀的正常工作。<br>（2）移动式压力容器设置的爆破片安全装置不应设置在安全阀的出口侧。                                                                                                                                                                   |
| ---- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 设置要求 | ① 爆破片与安全阀组合装置的泄放量应≥被保护承压设备的安全泄放量。<br>② 爆破片与安全阀之间的腔体应设置压力指示装置、排气口及合适的报警指示器。<br>③ 在爆破温度下，爆破片设计爆破压力与泄放管内存在的压力之和不应超过下列任一条件：<br>　a、安全阀的整定压力。<br>　b、在爆破片安全装置与安全阀之间的任何管路或管件的设计压力。<br>　c、被保护承压设备的设计压力。<br>④ 爆破片爆破后的泄放面积应足够大，以使流量与安全阀的额定排量相等。<br>⑤ 在爆破片以外的任何管道不应因爆破片爆破而被堵塞。 |
（3）爆破片安全装置与安全阀并联使用


| 设置情形 | ① 防止在异常工况下压力迅速升高的。  <br>② 作为辅助安全泄放装置，考虑在有可能遇到火灾或接近不能预料的外来热源需要增加泄放面积的。 |
| ---- | --------------------------------------------------------------------- |
| 泄放量  | 安全阀及爆破片各自的泄放量均应≥被保护承压设备的泄放量。 <br><br>                                 |
| 压力设置 | 爆破片的设计爆破压力应＞安全阀的整定压力。                                                 |
3、安装

| 爆破片的安装及管路设置 | ① 爆破片的泄放面积≤所接管路的截面积。<br>② 当有两个及两个以上爆破片安全装置采用排放汇集管时，汇集管的截面积应≥各爆破片安全装置出口管道截面积的总和。<br>③ 爆破片安全装置进口管应尽可能短、直，以免产生过大的压力损失。<br>④ 当爆破片的排放管中可能有可燃性介质排放时，应采取装设阻火器等预防措施，防止着火的危险。<br>⑤ 当爆破片安全装置的排放管中可能有毒性程度为中度的介质排放时，应装设辅助设施解除介质毒性后方可排出。<br>⑥ 换热器等承压设备，若高温介质有可能泄漏到低温介质而产生蒸气时，应在低温空间一侧设置爆破片安全装置。 |
| ----------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 组合安装        | ① 爆破片安全装置设置在承压设备和安全阀之间时，在安全阀入口侧应设置压力表、泄放阀等，以防止爆破片和安全阀间形成任何压力积聚。<br>② 安全阀设置在承压设备和爆破片安全装置之间时，在安全阀的出口侧应设置放空管、排液管等，以防止安全阀和爆破片装置之间形成压力积聚。<br>③ 爆破片安全装置和安全阀并联组合设置时，各自有引入管和引出管。                                                                                                           |
| 截止阀         | ① 当爆破片安全装置与承压设备之间设置截断阀，在正常工况下应保<br>持全开启状态并被锁住或铅封。<br>② 当需关闭截断阀，且承压设备仍在正常使用时，管理人员应留在现场。<br>③ 当管理人员离开现场，截断阀应处于全开启状态，并有铅封等防止关闭的措施。                                                                                                                                                    |
| 检查／更换       | ① 定期检查周期最长不得超过1年。<br>一般情况爆破片安全装置更换周期为2~3年，苛刻条件或重要场合下使用的爆破片应每年定期更换。<br>③ 设备检修中拆卸或设备长时间停工后（超过6个月），再次投入使用时，需立即更换。                                                                                                                                                                     |
五、化工机械设备的安全管理
1、常规检测
常规检测包括宏观检测和工量具检测等，是检查人员凭眼睛、手等感觉器官和简单工具等对设备进行检测，判别化工设备存在的外在缺陷。
2、无损检测

| 射线检测           | ① 对体型型缺陷（气孔、夹渣、夹钨、烧穿、咬边、焊瘤、凹坑等）检出率很高。（射击）                        |
| -------------- | ---------------------------------------------------------------- |
|                | ② 适宜检验厚度较薄的工件，不适宜检验较厚的工件，因为检验厚工件需要高能量的射线设备，而且随着厚度的增加，其检验灵敏度也会下降。 |
|                | ③ 检测成本高、速度慢。                                                     |
|                | ④ 具有辐射生物效应，损害生物组织、器官。                                            |
| 超声波检测          | 对面积型缺陷的检出率较高。（钟面）                                                |
|                | ② 检测成本低、速度快。                                                     |
|                | ③ 对人体及环境无害。                                                      |
| 磁粉检测           | ① 适用于检测铁磁性材料表面和近表面尺寸很小、间隙极窄，目视难以看出的缺陷。                           |
|                | ② 不能检测奥氏体不锈钢材料和用奥氏体不锈钢焊条焊接的焊缝，也不能检测铜、铝、镁、钛等非磁性材料。                |
|                | ③ 对于表面浅的划伤难以发现。                                                  |
| 渗透检测           | ① 可检测各种材料，金属、非金属材料，磁性、非磁性材料，以及焊接、锻造、轧制等加工方式。                     |
|                | ② 具有较高的灵敏度（可发现 0.1 μm 宽的缺陷）。                                     |
|                | ③ 检测费用低                                                          |
|                | ④ 只能检出表面开口的缺陷，难以确定缺陷的实际深度。<br>⑤ 不适于检查多孔性疏松材料制成的工件和表面粗糙的工件。       |
| 涡流检测           | ① 适用于导电材料。<br>② 操作速度很快。                                          |
| 超声波衍射时差法（TOFD） | ① 定量精度高。<br>② 适应性高于超声和射线。<br>③ 只需一人即可操作。<br>④ 成本低。               |
