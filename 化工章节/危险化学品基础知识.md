二.化学品类的标志、标签
1.[[安全标签]]的内容

| 化学品标识   | 用中文和英文分别标明化学品的化学名称或通用名称。名称要求醒目清晰，位于标签的上方。<br>对混合物应标出对其危险性分类有贡献的主要组分的化学名称或通用名、浓度或浓度范围。当需要标出的组分较多时，组分个数以不超过 5 个为宜。对于属于商业机密的成分可以不标明，但应列出其危险性。 |
| ------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| 象形图     |                                                                                                                                            |
| 信号词     | 用“危险”、“警告”两个词分别进行危害程度的警示。信号词位于化学品名称的下方。<br>危险用于严重的危险类别，警告用于较轻的类别。<br>存在多种危险性时，如果在安全标签上选用了信号词“危险”，则不应出现信号词“警告”                              |
| 危险性说明   | 简要概述化学品的危险特性，位于信号词下方                                                                                                                       |
| 防范说明    | 表述化学品在处置、搬运、储存和使用作业中所必须注意的事项和发生意外时简单有效的救护措施等，应包括安全预防措施、意外情况（如泄漏、人员接触或火灾等）的处理、安全储存措施及废弃处置等内容                                                |
| 供应商标识   | 供应商名称、地址、邮编和电话等                                                                                                                            |
| 应急咨询电话  | 填写化学品生产商或生产商委托的24h化学事故应急咨询电话。<br>国外进口化学品安全标签上应至少有一家中国境内的24h化学事故应急咨询电话                                                                      |
| 资料参阅提示语 | 提示化学品用户应参阅化学品安全技术说明书                                                                                                                       |

3.危险信息先后排序

| 象形图先后<br>顺序   | 如果使用了骷髅和交叉骨图形符号，则不应出现感叹号图形<br>符号                  |
| ------------- | ------------------------------------------------- |
|               | 如果使用了腐蚀图形符号，则不应出现感叹号来表示皮肤或<br>眼睛刺激                |
|               | 如果使用了呼吸致敏物的健康危害图形符号，则不应出现感<br>叹号来表示皮肤致敏物或者皮肤／眼睛刺激 |
| 信号词先后<br>顺序   | 存在多种危险性时，如果在安全标签上选用了信号词“危险”，<br>则不应出现信号词“警告”      |
| 危险性说明<br>先后顺序 | 所有危险性说明都应当出现在安全标签上，按物理危险、健<br>康危害、环境危害顺序排列        |
4.标签的设计

| 简化标签   | 对于≤100mL的化学品小包装[[危险化学品基础知识q3]]，为方便标签使用，安全标签要素可以简化，包括化学品标识、象形图、信号词、危险性说明、应急咨询电话、供应商名称及联系电话、资料参阅提示语即可。（少了“防范说明”） |
| ------ | -------------------------------------------------------------------------------------------------------------- |
| 标签内容编写 | 标签正文应使用简捷、明了、易于理解、规范的汉字表述，也可以同时使用少数民族文字或外文，但意义必须与汉字相对应，字形应小于汉字。                                                |
| 标签颜色   | 一般使用黑色图形符号加白色背景，方块边框为红色。正文应使用与底色反差明显的颜色，一般采用黑白色。若在国内使用，方块边框可以为黑色。                                              |
5.安全标签的使用

| 重复  | 若安全标签中的象形图与运输标志重复，安全标签中的象形图应删掉。                       |
| --- | ----------------------------------------------------- |
| 加贴  | 对组合容器，要求内包装加贴（挂）安全标签，外包装上加贴运输象形图，如果不需要运输标志可以加贴安全标签。   |
| 粘贴  | 安全标签应由生产企业在货物出厂前粘贴、挂拴、喷印。若要改换包装，则由改换包装单位重新粘贴、挂拴、喷印标签。 |
| 撕标签 | 盛装危险化学品的容器或包装，在经过处理并确认其危险性完全消除之后，方可撕下标签，否则不能撕下相应的标签。  |
6.安全标签的责任

| 生产企业 | 必须确保本企业生产的危险化学品在出厂时加贴符合国家标准的安全标签到危险化学品每个容器或每层包装上，使化学品供应和使用的每一阶段，均能在容器或包装上看到化学品的识别标志 |
| ---- | ----------------------------------------------------------------------------------- |
| 使用单位 | 使用的化学危险品应有安全标签，并应对包装上的安全标签进行核对。若安全标签脱落或损坏时，经检查确认后应立即补贴                              |
| 经销单位 | 经销的危险化学品必须具有安全标签，进口的危险化学品必须具有符合我国标签标准的中文安全标签                                        |
| 运输单位 | 对无安全标签的危险品一律不能承运                                                                    |
三.化学品的安全技术说明书
1、化学品安全技术说明书（[[SDS]]）的内容
　　　　　

| 化学品及<br>企业标识      | 主要标明化学品名称、生产企业名称、地址、邮编、电话、应急电话、<br>传真和电子邮件地址等信息                                                                             |
| ----------------- | --------------------------------------------------------------------------------------------------------------------------- |
| 危险性<br>概述         | 主要包括：危害类别、侵入途径、健康危害、环境危害、燃爆危险等信息。侵入途径包括皮肤、呼吸道、消化道。                                                                          |
| 成分，组<br>成信息       | 标明该化学品是纯化学品还是混合物。<br>纯化学品：应给出其化学品种名称或商品名和通用名。<br>混合物：应给出危害性组分的浓度或浓度范围。<br>无论是纯化学品还是混合物，如果其中包含有害性组分，则应给出<br>化学文摘索引登记号（CAS 号） |
| 急救措施              | 急救措施指作业人员意外地受到伤害时，所需采取的现场自救或<br>互救的简要处理方法，包括眼睛接触、皮肤接触、吸入、食入的<br>急救措施。                                                       |
| 消防措施              | 主要表示化学品的物理和化学特殊危险性，适合灭火介质，不合<br>适的灭火介质以及消防人员个体防护等方面的信息，包括：危险<br>特性、灭火介质和方法，灭火注意事项等。                                         |
| 泄漏应急处理            | 泄漏应急处理指化学品泄漏后现场可采用的简单有效的应急措施。<br>注意事项和消除方法，包括：应急行动、应急人员防护、环保措施、<br>消除方法等内容。                                                 |
| 操作处置<br>与储存       | 主要是指化学品操作处置和安全储存方面的信息资料，包括操作<br>处置作业中的安全注意事项、安全储存条件和注意事项。                                                                   |
| 接触控<br>制、个体<br>防护 | 在生产、操作处置、搬运和使用化学品的作业过程中，为保护作业人员免受化学品危害而采取的防护方法和手段。包括最高容许浓度、工程控制、呼吸系统防护、眼睛防护、身体防护、手防护、其他防护要求。                                |
| 理化特性              | 主要描述化学品的外观及理化性质等方面的信息，包括外观与性状、pH、沸点、熔点、相对密度、相对蒸气密度、饱和蒸气压、燃烧热、临界温度、临界压力、辛醇／水分配系数、闪点、引燃温度、爆炸极限、溶解性、主要用途和其他一些特殊理化性质            |
| 稳定性和反应性           | 主要叙述化学品的稳定性和反应活性方面的信息，包括稳定性、禁配物、应避免接触的条件、聚合危害、分解产物                                                                          |
| 毒理学<br>资料         | 毒性、刺激性、致敏性、亚急性和慢性毒性，致突变性、致畸性、致癌性等                                                                                           |
| 生态学<br>资料         | 主要陈述化学品的环境生态效应、行为和转归，包括生物效应、生物降解性、生物富集、环境迁移及其他有害的环境影响等                                                                      |
| 废弃处置              | 废弃处置是指对被化学品污染的包装和无使用价值的化学品的安全处理方法，包括废弃处置方法和注意事项                                                                             |
| 运输信息              | 主要是指国内、国际化学品包装、运输的要求及运输规定的分类和编号，包括：危险货物编号、包装类别、包装标志、包装方法、UN 编号及运输注意事项等                                                      |
| 法规信息              | 主要是化学品管理方面的法律条款和标准                                                                                                          |
| 其他信息              | 主要提供其他对安全有重要意义的信息，包括：参考文献、填表时间、填表部门、数据审核单位等                                                                                 |
2、化学品安全技术说明书编写和使用要求[[危险化学品基础知识q4]]
（1）安全技术说明书规定的十六大项内容在编写时不能随意删除或合并，其顺序不可随意变更。
（2）安全技术说明书的内容，从该化学品的制作之日算起，每5年更新一次，若发现新的危害性，在有关信息发布后的半年内，生产企业必须对安全技术说明书的内容进行修订。
（3）安全技术说明书采用“一个品种一卡”的方式编写，同类物、同系物的技术说明书不能互相替代。混合物要填写有害性组分及其含量范围。
