| 安全附件 | 要明确火炬排放系统、报警仪、安全阀、压力表、防爆门、消防气防设施、风向标等安全附件的检查、投用时间。      |

 二、开停工过程安全技术要点
1、停工过程注意事项
（1）降温降量的速度不宜过快，尤其在高温条件下，温度骤变会造成设备和管道变形、破裂，引起易燃易爆、有毒介质泄漏而发生着火爆炸或人员中毒。
（2）开关阀门的操作要缓慢，开蒸汽阀门时要注意管线的预热、排凝和防水击。
（3）停炉操作按照降温曲线进行，注意各部位火嘴熄火对炉膛降温均匀性的影响。
（4）装置停车时，系统内的物料应尽可能倒空、抽净，降温后送出装置，可燃、有毒气体应排至火炬烧掉，对残存物料的排放，不得就地排放或排入下水道中。 
2、吹扫

| 介质  | 一般采用蒸汽或惰性气体进行吹扫。                              |
| --- | --------------------------------------------- |
| 酸碱  | 存放酸碱介质的设备、管线，应先予以中和或加水冲洗。                     |
| 排液  | 低沸点物料倒空置换一定要先排液后放压，防止液态烃排放过程大量汽化，使管线设备冷脆断裂。   |
| 阀门  | 吹扫合格后，应先关闭有关阀门，再停气，以防止系统介质倒回，同时及时加盲板与有物料系统隔离。 |
3、置换

| 介质   | 大多采用蒸汽、氮气等惰性气体为置换介质，也可采用注水排气法。                                          |
| ---- | ----------------------------------------------------------------------- |
| 取样口  | 介质密度                   取样口                    置换介质口                     |
|      | 置换＞被置换           顶部＋死角              底部                                  |
|      | 置换＜被置换           底部＋死角              顶部                                  |
| 尾气   | 置换出的可燃、有毒气体，应排至火炬烧掉。                                                    |
| 气体分析 | 用惰性气体置换过的设备，若需进入其内部作业，还必须采用自然通风或强制通风的方法将惰性气体置换掉，化验分析合格后方可进入作业，以防窒息事故发生。 |
 4、蒸煮和清洗

| 水溶性物质 | 可用水洗或热水蒸煮。                                                                  |
| ----- | --------------------------------------------------------------------------- |
| 粘稠性物料 | 可先用蒸汽吹扫，再用热水洗求。                                                             |
| 特殊    | 不溶于水或在安全上有特殊要求的积附物，可用化学清洗。化学<br><br>清洗时，应注意采取措施防止可能产生的硫化氢等有毒气体<br><br>危害人体。 |

四、开工条件联合检查确认


| 新建生产装置或大修装置 | ① 中间交接前进行“三查四定”<br>② 装置引料前进行开工条件检查 |
| ----------- | ---------------------------------- |
| 单体设备设施      | 开工条件检查                             |
1、三查四定
查设计漏洞、查工程质量及隐患、查未完工程量，定任务、定人员、定时间、定措施。
2、中间交接的一般要求
按照中间交接的标准进行三查四定并完成问题整改后，就可以进行中间交接，要求：
（1）工程按设计内容完成施工。
（2）工程质量符合国家和行业标准及工程项目合同中约定的内容。
（3）工艺、动力管道的耐压试验完成，系统清洗、吹扫完成，保温基本完成。
（4）静设备无损检验、强度试验、内件安装、清扫完成；安全附件（安全阀、防爆门等）已调试合格。
（5）转动设备单机试车合格。
（6）大型机组空负荷试车完成，机组仪表、保护性联锁和报警等自控系统调试联合合格。
（7）装置电气、仪表、计算机、防毒防火防爆等调试联校合格。
（8）装置区施工临时设施拆除，完工、干净、场地清，竖向工程施工完成。
（9）对联动试运行有影响的“三查四定”项目及设计变更项处理完，其他未施工尾项责任、完成时间已明确。

4、开工过程安全管理
一般步骤操作员自己确认，关键步骤由工程技术人员进行确认。

| 开工前   | ① 已经对员工进行了开工方案的 培训和考试，员工已熟练掌握。<br>② 已经完成系统试压并做到气密性良好。<br>③ 系统的空气已经彻底吹扫完成、置换干净合格，不会产生爆炸性混合气体。<br>④ 所有管线、设备上的盲板已经全部拆除并做到编号和记录。                                                                                                                                                                                                                                |
| ----- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 开工过程中 | ① 使用蒸汽前已经脱净冷凝水。<br>② 在开工过程中严密监测加热炉的负荷变化，严格执行加热炉的操作规程。<br>③ 操作人员认真对待各类报警信号，对所发生的各种报警认真检查确认。<br>④ 当管线、容器内有介质存在时，不带压进行更换垫片、压盘根等操作。<br>⑤ 各类塔、容器恒温脱水时操作人员不得离开。<br>⑥ 针对开工阶段产生毒物的过程，现场人员全部佩戴合适的空气呼吸器和便携式有毒气体检测报警仪。<br>⑦ 开工阶段，各专业人员、各级人员认真巡回检查，及时发现并处理出现的问题。<br>⑧ 保运单位配备足够的保运人员，随时处理开工过程中出现的各种设备、电气、仪表故障。<br>⑨ 开工完成后的运行初期，装置工况还不稳定，建议开展一次综合大检查，以及时发现可能存在的技术问题和安全隐患。 |
