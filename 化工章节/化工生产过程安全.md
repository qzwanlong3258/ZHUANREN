2.试生产安全管理

| 系统吹扫冲洗安全管理 | 在系统吹扫前，要在排放口设置警戒区，拆除易被吹扫冲洗损坏的所有部件，蒸汽吹扫时，要落实防止人员烫伤的防护措施。[[化工生产过程安全q3]]                                                                                                                |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 气密实验安全管理   | 1.要确保气密实验全覆盖，无遗漏，明确各系统气密的最高压力等级。2.高压系统气密实验前，要分成若干等级压力，逐级进行气密实验。3.真空系统进行真空实验前，要先完成气密试验。4.要用盲板将气密试验与其他系统隔离，严禁超压。5.气密试验时，要安排专人监控，发现问题，及时处理，做好气密检查记录，签字备查。[[化工生产过程安全q5]]                 |
| 单机试车安全管理   | 1.企业要建立单机试车安全管理程序。2.单机试车前，要编制试车方案，操作规程，并经各专业确认。3.单机试车过程中，应安排专人操作，监控，记录，发现异常立即处理。4.单机试车结束后，建设单位要组织设计，施工，监理及制造商等方面人员签字确认并填写试车记录。[[化工生产过程安全q4]]                                         |
| 联动试车安全管理   | 一，联动试车应具备下列条件：1.所有操作人员考核合格并已取得上岗资格；2.公用工程系统已稳定运行；3.试车方案和相关操作规程，经审查批准的仪表报警和联锁值已整定完毕。4.各类生产记录，报表已印发到岗位；负责统一指挥的协调人员已经确定。二，引入燃料或窒息性气体后，企业必须建立并执行每日安全调度例会制度，统筹协调全部试车的安全管理工作[[化工生产过程安全q6]] |
| 投料安全管理     | 1.投料前，要全面检查工艺，设备，电气，仪表，公用工程和应急准备等情况，具备条件后方可进行投料。2.投料及试生产过程中，管理人员要现场指挥，操作人员要持续进行现场巡查，设备，电气，仪表等专业人员要加强现场巡检，发现问题及时报告和处理。3.投料试生产过程中，要严格控制现场人数，严禁无关人员进入现场。[[化工生产过程安全q7]]                  |
3.承包商管理[[化工生产过程安全q8]]
1. 企业要建立承包商安全管理制度，将承包商在本企业发生的事故纳入企业事故管理。
2. 企业要严格审查承包商有关资质，定期评估承包商安全生产业绩，及时淘汰业绩差的承包商。
3. 企业要对承包商作业人员进行严格的入厂安全教育培训，经考核合格的方可凭证入场。
4. 承包商进入作业现场前，企业要与承包商作业人员进行现场安全交底，审查承包商编制的施工方案和作业安全措施。
5. 企业要与承包商签订安全管理协议，明确双方安全管理范围和职责
6. 企业要对承包商进行全程安全监督
4.变更管理
变更管理的分类[[化工生产过程安全q9]] 

| 工艺技术变更 | 主要包括生产能力，原辅材料（包括助剂，添加剂，催化剂等）和介质（包括成分比例的变化），工艺路线，流程及操作条件，工艺操作规程或操作方法，工艺控制参数，仪表控制系统（包括安全报警和联锁整定值的改变），水，电，汽，风等公用工程方面的改变。 |
| ------ | --------------------------------------------------------------------------------------------------------------------- |
| 设备设施变更 | 主要包括设备设施的更新改造，非同类型替换（包括型号，材质，安全设施的变更），布局改变，备件，材料的改变，监控，测量仪表的变更，计算机及软件的变更，电气设备的变更，增加临时的电气设备等                           |
| 管理变更   | 主要包括人员，供应商和承包商，管理机构，管理职责，管理制度和标准发生变化等。注：仪表控制系统设定值改变值属于工艺技术变更，仪表本身的变更属于设备设施变更。                                         |
变更管理程序[[化工生产过程安全q10]] 

| 申请    | 按要求填写变更申请表。由专人进行管理。                                                            |
| ----- | ------------------------------------------------------------------------------ |
| 风险评估  | 应采用合适的危害辨识和风险评估办法开展变更风险评估，制定管控措施                                               |
| 审批    | 变更申请表应逐级上报企业主管部门，并按管理权限报主管负责人审批                                                |
| 实施    | 变更批准后，由企业主管部门负责实施。没有经过审查和批准，任何临时性变更都不得超过原批准范围和期限                               |
| 验收，关闭 | 变更结束后，企业主管部门应对变更实施情况进行验收并形成报告，及时通知相关部门和有关人员，相关部门收到变更验收报告，要及时更新安全生产信息，载入变更管理档案。 |
